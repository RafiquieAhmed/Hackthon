what is Table?

In HTML, a table is a structured way to display data in rows and columns. It allows web authors to organize information like text, images, links, or even other tables, into a grid-like format for better readability and understanding

The core elements used to create an HTML table are:

  • <table>This is the main container tag that defines the entire table.
  • <tr> This tag defines a table row, which is a horizontal grouping of cells.
  • <th> This tag defines a table header cell, typically used in the first row or column to label the data. By default, content within <th> is bold and centered.
  • <td>This tag defines a standard table data cell, containing the actual data for each row.

Additionally, tables can include:

  • <caption> Provides a descriptive caption for the table, usually appearing at the top.
  • <thead> <tbody> <tfoot> These tags are used to semantically group the table's header, body, and footer sections, respectively.
  • <colspan> <rowspan>These attributes can be used within <th> or <td> tags to make a cell span across multiple columns or rows, effectively merging them.

Deployment and media tags

HTML MEDIA

First Principle

  • The web can show more than text and images.
  • Older browsers used plugins (Flash, QuickTime), which were insecure & inconsistent.
  • HTML5 introduced <video> and <audio>and tags to embed media natively without plugins.
  • Attribute of video src
    • width/height : set player size
    • autoplay + muted : play automatically (muted needed due to browser policy)
    • loop : restart when finished
    • poster :thumbnail image shown before play

    Audio Tag

  • Works just like <video> but without visuals.
  • Supports attributes like autoplay, loop, muted.

    • code:

    <audio src="audio/song.mp3" controls> </audio>
